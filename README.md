# TrendPulse - Task 1 (Data Collection)

## Objective
Fetch trending stories from HackerNews API and categorize them.

## Features
- Fetch top 500 stories
- Categorize into 5 groups:
  - technology
  - worldnews
  - sports
  - science
  - entertainment
- Collect up to 25 per category
- Save results as JSON

## Output
Stored in:
data/trends_YYYYMMDD.json

## Libraries Used
- requests
- json
- time
- datetime
